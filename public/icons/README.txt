PLACEHOLDER ICONS - Replace before production
=============================================
Required files:
  - icon-192x192.png   (192x192 px, PNG)
  - icon-512x512.png   (512x512 px, PNG)

A reference SVG is provided: icon-placeholder.svg
Use it to generate the PNGs once a real logo is ready.

Tools to convert:
  - https://realfavicongenerator.net
  - Figma / Illustrator export
  - `sharp` npm package for CLI conversion
